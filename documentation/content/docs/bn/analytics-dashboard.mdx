## এটা কী করে

অ্যানালিটিক্স ড্যাশবোর্ড মেসেজ ডেলিভারি, রিড রেট এবং রিপ্লাই রেটের প্রি-অ্যাগ্রিগেটেড পরিসংখ্যান দেখায় — সব ক্যাম্পেইন জুড়ে ৩০ দিনের রোলিং উইন্ডোতে।

## ড্যাশবোর্ড সেকশন

### ওভারভিউ (গত ৩০ দিন)

| মেট্রিক | সংজ্ঞা |
|---------|--------|
| পাঠানো | `SENT` স্ট্যাটাসে পৌঁছানো মেসেজ |
| ডেলিভার | প্রাপকের ফোনে পৌঁছানো |
| পঠিত | প্রাপক দ্বারা খোলা |
| ব্যর্থ | ডেলিভার করা যায়নি |
| ডেলিভারি রেট | `deliveredCount / sentCount × 100` |
| রিড রেট | `readCount / deliveredCount × 100` |

### ক্যাম্পেইন অ্যানালিটিক্স

একক ক্যাম্পেইনের জন্য একই মেট্রিক, সাথে:
- **ডেলিভারি ফানেল:** মোট → পাঠানো → ডেলিভার → পঠিত → রিপ্লাই
- **প্রতি-কন্টাক্ট স্ট্যাটাস টেবিল**

## রিপোর্ট এক্সপোর্ট

### এক্সপোর্ট শুরু

```json
POST /api/v1/analytics/campaigns/:id/export
{ "format": "csv" }
```

### স্ট্যাটাস চেক

```
GET /api/v1/analytics/exports/:jobId
```

| স্ট্যাটাস | অর্থ |
|----------|-------|
| `PENDING` | জব কিউতে |
| `PROCESSING` | ফাইল তৈরি হচ্ছে |
| `COMPLETE` | ফাইল প্রস্তুত (২৪ ঘণ্টায় ডাউনলোড) |
| `FAILED` | ব্যর্থ; পুনরায় চেষ্টা করুন |

## API এন্ডপয়েন্ট

```
GET  /api/v1/analytics/overview                      — ৩০ দিনের সামগ্রিক পরিসংখ্যান
GET  /api/v1/analytics/campaigns                     — ক্যাম্পেইন তালিকা
GET  /api/v1/analytics/campaigns/:id                 — একক ক্যাম্পেইন
GET  /api/v1/analytics/campaigns/:id/contacts        — প্রতি-কন্টাক্ট স্ট্যাটাস
POST /api/v1/analytics/campaigns/:id/export          — এক্সপোর্ট শুরু
GET  /api/v1/analytics/exports/:jobId                — এক্সপোর্ট স্ট্যাটাস
```
