## এটা কী করে

ইনবক্সে প্রতিটি কন্টাক্টের সাথে সব ইনবাউন্ড ও আউটবাউন্ড মেসেজ একসাথে দেখা যায়। সম্পূর্ণ মেসেজ থ্রেড পড়তে এবং ক্যাম্পেইন ছাড়াই সরাসরি ম্যানুয়াল রিপ্লাই পাঠাতে পারবেন।

## ইনবক্স কাঠামো

### কথোপকথন তালিকা

প্রতি কন্টাক্টে একটি রো, সাম্প্রতিক মেসেজ অনুযায়ী সাজানো:

| ফিল্ড | বিবরণ |
|--------|--------|
| কন্টাক্ট নাম / ফোন | E.164 ফোন নম্বর দ্বারা চিহ্নিত |
| শেষ মেসেজ প্রিভিউ | শেষ মেসেজের সংক্ষিপ্ত বডি |
| দিক | `INBOUND` বা `OUTBOUND` |
| অপঠিত সংখ্যা | অপঠিত ইনবাউন্ড মেসেজ সংখ্যা |

তালিকা প্রতি **১০ সেকেন্ডে** অটো-রিফ্রেশ হয়।

## ধাপসমূহ

### ১. সব কথোপকথন দেখুন

```
GET /api/v1/inbox?limit=50&offset=0
```

### ২. থ্রেড খুলুন

```
GET /api/v1/inbox/:contactId?limit=50
```

### ৩. পুরনো মেসেজ লোড (কার্সর পেজিনেশন)

```
GET /api/v1/inbox/:contactId?cursor=<messageId>&limit=50
```

### ৪. ম্যানুয়াল রিপ্লাই পাঠান

```json
POST /api/v1/inbox/:contactId/send
{
  "sessionId": "wa-session-uuid",
  "body": "আপনার অর্ডারটি আগামীকাল ডেলিভারি হবে।"
}
```

## রিয়েল-টাইম আপডেট

WebSocket এ নতুন ইনবাউন্ড মেসেজ আসে:

```
message:inbound  { contactId, message: { id, body, direction, status, createdAt } }
```

## API এন্ডপয়েন্ট

```
GET  /api/v1/inbox                      — কথোপকথন তালিকা
GET  /api/v1/inbox/:contactId           — মেসেজ থ্রেড
POST /api/v1/inbox/:contactId/send      — ম্যানুয়াল মেসেজ পাঠান
```
