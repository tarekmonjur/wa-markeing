## ভেরিয়েবল সিনট্যাক্স

টেমপ্লেটে ডাবল কার্লি ব্রেসে ভেরিয়েবল ব্যবহার করা যায়:

| ভেরিয়েবল | মান |
|----------|------|
| `{{name}}` | কন্টাক্টের নাম |
| `{{phone}}` | কন্টাক্টের ফোন (লোকাল ফরম্যাট) |
| `{{email}}` | কন্টাক্টের ইমেইল |
| `{{date}}` | বর্তমান তারিখ (DD/MM/YYYY) |
| `{{time}}` | বর্তমান সময় (HH:MM) |
| `{{custom.fieldName}}` | কন্টাক্টের কাস্টম ফিল্ড |

### উদাহরণ টেমপ্লেট

```
আস্সালামু আলাইকুম {{name}} ভাই! 🌙
ঈদ উপলক্ষে সকল পণ্যে ২০% ছাড়!
শোরুম: {{custom.city}}
```

## সীমাবদ্ধতা

- সর্বোচ্চ মেসেজ দৈর্ঘ্য: **৪০৯৬ অক্ষর**
- HTML ট্যাগ স্বয়ংক্রিয়ভাবে মুছে যায়

## প্রিভিউ

পাঠানোর আগে ভেরিয়েবল রেজোলিউশন টেস্ট করুন:

```
POST /api/v1/templates/:id/preview
Body: { "contactId": "uuid" }
```

## API এন্ডপয়েন্ট

```
POST   /api/v1/templates              — টেমপ্লেট তৈরি
GET    /api/v1/templates              — টেমপ্লেট তালিকা
GET    /api/v1/templates/:id          — টেমপ্লেট দেখুন
PATCH  /api/v1/templates/:id          — টেমপ্লেট আপডেট
DELETE /api/v1/templates/:id          — টেমপ্লেট মুছুন
POST   /api/v1/templates/:id/preview  — কন্টাক্ট দিয়ে প্রিভিউ
```
