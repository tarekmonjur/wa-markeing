## এটা কী করে

AI কপি জেনারেটর আপনার ব্যবসার প্রসঙ্গের ভিত্তিতে সংক্ষিপ্ত, আকর্ষণীয় হোয়াটসঅ্যাপ মার্কেটিং মেসেজ তৈরি করে। দুটি AI প্রোভাইডার সাপোর্টেড: **OpenAI** (ক্লাউড) এবং **Ollama** (লোকাল, বিনামূল্যে)।

## পূর্বশর্ত

- **OpenAI:** এনভায়রনমেন্টে `OPENAI_API_KEY`
- **Ollama:** Docker এ `ollama` সার্ভিস এবং `llama3.2:3b` মডেল
  ```bash
  docker exec ollama ollama pull llama3.2:3b
  ```

## ধাপসমূহ

### ১. ফর্ম পূরণ করুন

| ফিল্ড | আবশ্যক | উদাহরণ |
|--------|--------|---------|
| `businessName` | হ্যাঁ | `ঢাকা ফ্যাশন হাউস` |
| `product` | হ্যাঁ | `ঈদ কালেকশন ২০২৬` |
| `goal` | হ্যাঁ | `২০% ছাড় ঘোষণা, অনলাইন অর্ডার বাড়ানো` |
| `tone` | হ্যাঁ | `friendly`, `professional`, `urgent` |

### ২. কপি জেনারেট

```json
POST /api/v1/ai/generate
{
  "businessName": "সিলেট ফুডস",
  "product": "ইফতার কম্বো মিল",
  "goal": "রমজানের ইফতার ডিল প্রমোট",
  "tone": "friendly"
}
```

### ৩. রেসপন্স

```json
{
  "copy": "ইফতারের সেরা অফার! 🌙 সিলেট ফুডস-এর ইফতার কম্বো এখন মাত্র ২৯৯ টাকায়।",
  "provider": "openai",
  "remaining": 8
}
```

## AI প্রোভাইডার তুলনা

| প্রোভাইডার | মডেল | খরচ | মান | সময় |
|-----------|-------|------|------|------|
| OpenAI | `gpt-4o-mini` | প্রতি টোকেন | উচ্চ (EN + BN) | ~১–৩ সেকেন্ড |
| Ollama | `llama3.2:3b` | বিনামূল্যে | ভালো (EN) | ৫–১৫ সেকেন্ড |

## নিরাপত্তা — প্রম্পট ইনজেকশন প্রতিরোধ

ইনপুট স্বয়ংক্রিয়ভাবে স্যানিটাইজ করা হয়। ক্ষতিকর প্যাটার্ন মুছে দেওয়া হয়।

## সমস্যা সমাধান

| সমস্যা | কারণ | সমাধান |
|--------|------|--------|
| `429 Too Many Requests` | কোটা শেষ | মধ্যরাত পর্যন্ত অপেক্ষা বা প্ল্যান আপগ্রেড |
| Ollama ব্যবহার হচ্ছে | OpenAI কী নেই | `.env` তে `OPENAI_API_KEY` সেট করুন |
