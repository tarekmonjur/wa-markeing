## এটা কী করে

একাধিক হোয়াটসঅ্যাপ অ্যাকাউন্ট (ফোন নম্বর) সংযোগ করুন এবং প্রতিটি ক্যাম্পেইনে নির্দিষ্ট অ্যাকাউন্ট নির্ধারণ করুন। ব্র্যান্ড, অঞ্চল বা কন্টাক্ট তালিকা অনুযায়ী পাঠানো আলাদা করুন।

## সেশন স্ট্যাটাস

| স্ট্যাটাস | অর্থ |
|----------|-------|
| `DISCONNECTED` | সংযুক্ত নয় |
| `CONNECTING` | সকেট শুরু হচ্ছে |
| `QR_READY` | QR প্রদর্শিত (৬০ সেকেন্ড) |
| `CONNECTED` | পেয়ার ও অনলাইন |
| `TOS_BLOCK` | হোয়াটসঅ্যাপ ToS ফ্ল্যাগ |
| `BANNED` | স্থায়ীভাবে ব্যান |

## প্ল্যান অনুযায়ী সীমা

| প্ল্যান | সর্বোচ্চ সেশন |
|---------|--------------|
| FREE | ১ |
| STARTER | ২ |
| PRO | ৫ |
| AGENCY | ২০ |

## ব্যান প্রতিরোধ সেরা অভ্যাস

| নির্দেশিকা | কারণ |
|-----------|-------|
| প্রতি অ্যাকাউন্টে দৈনিক সর্বোচ্চ ২০০ মেসেজ | প্ল্যাটফর্ম দ্বারা প্রয়োগকৃত |
| মেসেজের মধ্যে ৩-৮ সেকেন্ড | মানুষের আচরণ অনুকরণ |
| ভিন্ন টেমপ্লেট ব্যবহার | একই বাল্ক মেসেজ = ব্যান সংকেত |
| `TOS_BLOCK` এ পজ করুন | চালিয়ে গেলে পূর্ণ ব্যান হবে |

## API এন্ডপয়েন্ট

```
POST   /api/v1/whatsapp/sessions         — সেশন তৈরি
GET    /api/v1/whatsapp/sessions         — তালিকা
PATCH  /api/v1/whatsapp/sessions/:id     — লেবেল/ডিফল্ট আপডেট
DELETE /api/v1/whatsapp/sessions/:id     — সংযোগ বিচ্ছিন্ন
GET    /api/v1/whatsapp/sessions/:id/qr  — QR কোড (SSE)
```
