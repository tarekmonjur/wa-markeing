## এটা কী করে

মেসেজ টেমপ্লেটে ছবি, ভিডিও, অডিও ক্লিপ এবং ডকুমেন্ট অ্যাটাচ করুন। ক্যাম্পেইন চালালে মিডিয়া ও টেমপ্লেট টেক্সট একসাথে পাঠানো হয়।

## সাপোর্টেড ফাইল টাইপ

| ক্যাটাগরি | MIME টাইপ | সর্বোচ্চ সাইজ |
|----------|-----------|--------------|
| **ইমেজ** | `image/jpeg`, `image/png`, `image/webp`, `image/gif` | ১৬ MB |
| **ভিডিও** | `video/mp4`, `video/3gpp` | ১০০ MB |
| **অডিও** | `audio/aac`, `audio/mp4`, `audio/mpeg` | ১৬ MB |
| **ডকুমেন্ট** | `application/pdf`, Word, Excel | ১০০ MB |

## আপলোড ফ্লো

```
১. POST /api/v1/media/upload  (multipart/form-data)
২. সার্ভার ম্যাজিক বাইটস দিয়ে MIME যাচাই করে
৩. ফাইল MinIO তে স্ট্রিম করা হয়
৪. রেসপন্স: { mediaId, url, type, mimeType, size }
```

## ধাপসমূহ

### ১. ফাইল আপলোড

```bash
curl -X POST http://localhost:3001/api/v1/media/upload \
  -H "Authorization: Bearer <token>" \
  -F "file=@./eid-offer.jpg"
```

### ২. টেমপ্লেটে মিডিয়া যুক্ত করুন

```json
POST /api/v1/templates
{
  "name": "ঈদ অফার ব্যানার সহ",
  "body": "আস্সালামু আলাইকুম {{name}}! ঈদ উপলক্ষে ২০% ছাড়।",
  "mediaUrl": "https://minio.../wa-media/eid-offer.jpg",
  "mediaType": "IMAGE"
}
```

## নিরাপত্তা — এক্সটেনশন স্পুফিং প্রতিরোধ

সার্ভার **ম্যাজিক বাইটস** দিয়ে ফাইলের প্রকৃত কন্টেন্ট যাচাই করে, ফাইল এক্সটেনশন নয়।

## API এন্ডপয়েন্ট

```
POST /api/v1/media/upload      — ফাইল আপলোড
GET  /api/v1/media/:mediaId    — নতুন presigned URL
```

## সমস্যা সমাধান

| সমস্যা | কারণ | সমাধান |
|--------|------|--------|
| `415 Unsupported Media Type` | ফাইল টাইপ অনুমোদিত নয় | সাপোর্টেড টাইপ দেখুন |
| `413 Payload Too Large` | সাইজ সীমা অতিক্রম | কমপ্রেস বা রিসাইজ করুন |
| URL মেয়াদ শেষ | ১ ঘণ্টার বেশি পুরনো | `GET /media/:mediaId` কল করুন |
