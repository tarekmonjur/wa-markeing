## এটা কী করে

সহকর্মীদের আপনার অ্যাকাউন্টে আমন্ত্রণ জানান এবং তারা কী করতে পারবে তা নিয়ন্ত্রণ করুন।

## রোল

| রোল | কার জন্য |
|------|---------|
| `ADMIN` | অ্যাকাউন্ট মালিক ও বিশ্বস্ত ম্যানেজার — সম্পূর্ণ নিয়ন্ত্রণ |
| `AGENT` | সেলস বা সাপোর্ট স্টাফ — পাঠাতে ও দেখতে পারে |
| `VIEWER` | স্টেকহোল্ডার, ক্লায়েন্ট — শুধু পড়তে পারে |

## পারমিশন ম্যাট্রিক্স

| কার্যকলাপ | ADMIN | AGENT | VIEWER |
|----------|-------|-------|--------|
| ড্যাশবোর্ড ও অ্যানালিটিক্স দেখা | ✅ | ✅ | ✅ |
| কন্টাক্ট, টেমপ্লেট, ক্যাম্পেইন দেখা | ✅ | ✅ | ✅ |
| ইনবক্স দেখা | ✅ | ✅ | ✅ |
| কন্টাক্ট ও টেমপ্লেট তৈরি/সম্পাদনা | ✅ | ✅ | ❌ |
| কন্টাক্ট ইম্পোর্ট | ✅ | ✅ | ❌ |
| ক্যাম্পেইন তৈরি/শুরু | ✅ | ✅ | ❌ |
| ম্যানুয়াল রিপ্লাই | ✅ | ✅ | ❌ |
| মুছে ফেলা | ✅ | ❌ | ❌ |
| হোয়াটসঅ্যাপ সেশন পরিচালনা | ✅ | ❌ | ❌ |
| টিম সদস্য পরিচালনা | ✅ | ❌ | ❌ |

নিষিদ্ধ কাজ **`403 Forbidden`** রিটার্ন করে।

## API এন্ডপয়েন্ট

```
GET    /api/v1/teams/members             — সদস্য তালিকা
POST   /api/v1/teams/members/invite      — আমন্ত্রণ (ADMIN)
PATCH  /api/v1/teams/members/:id         — রোল আপডেট (ADMIN)
DELETE /api/v1/teams/members/:id         — সদস্য অপসারণ (ADMIN)
```
