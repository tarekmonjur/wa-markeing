## সারসংক্ষেপ

হোয়াটসঅ্যাপ সেশন `/api/v1/whatsapp/sessions` এন্ডপয়েন্টের মাধ্যমে পরিচালিত হয়। প্রতিটি ব্যবহারকারী এক বা একাধিক সেশন তৈরি করতে পারেন যা Baileys সেশন ম্যানেজার সার্ভিসের সাথে সংযুক্ত হবে।

## সেশন লাইফসাইকেল

```
তৈরি → QR স্ক্যান → সংযুক্ত → (সংযোগ বিচ্ছিন্ন)
```

1. **তৈরি** — `POST /api/v1/whatsapp/sessions` নতুন সেশন রেকর্ড তৈরি করে
2. **QR স্ক্যান** — সেশন ম্যানেজার QR কোড তৈরি করে পেয়ারিং এর জন্য
3. **সংযুক্ত** — স্ক্যান করলে স্ট্যাটাস `CONNECTED` হয়
4. **সংযোগ বিচ্ছিন্ন** — ফোন অফলাইন হলে স্ট্যাটাস `DISCONNECTED` হয়

## সেশন স্ট্যাটাস

| স্ট্যাটাস | বিবরণ |
|----------|--------|
| `DISCONNECTED` | প্রাথমিক অবস্থা, পেয়ার করা হয়নি |
| `CONNECTING` | QR কোড প্রদর্শিত, স্ক্যানের অপেক্ষায় |
| `CONNECTED` | সফলভাবে পেয়ার করা ও অনলাইন |
| `LOGGED_OUT` | ব্যবহারকারী নিজে লগ আউট করেছেন |

## কিভাবে সংযোগ করবেন

1. ড্যাশবোর্ডে **সেটিংস → হোয়াটসঅ্যাপ** এ যান
2. **অ্যাকাউন্ট যোগ করুন** ক্লিক করুন
3. সেশনের জন্য একটি লেবেল দিন (যেমন: "সেলস টিম")
4. একটি QR কোড দেখাবে — আপনার হোয়াটসঅ্যাপ অ্যাপ দিয়ে স্ক্যান করুন:
   - হোয়াটসঅ্যাপ → সেটিংস → লিংকড ডিভাইস
   - "ডিভাইস লিংক করুন" ট্যাপ করুন
   - QR কোড স্ক্যান করুন
5. স্ক্যান করলে স্ট্যাটাস `CONNECTED` হয়ে যাবে

QR কোড **৬০ সেকেন্ড** পর মেয়াদ শেষ হয়। মেয়াদ শেষ হলে **QR রিফ্রেশ** ক্লিক করুন।

## API এন্ডপয়েন্ট

```
POST   /api/v1/whatsapp/sessions      — সেশন তৈরি
GET    /api/v1/whatsapp/sessions      — সব সেশন দেখুন
GET    /api/v1/whatsapp/sessions/:id  — সেশনের বিস্তারিত
DELETE /api/v1/whatsapp/sessions/:id  — সেশন মুছুন
```

## সমস্যা সমাধান

| সমস্যা | কারণ | সমাধান |
|--------|------|--------|
| QR দেখা যাচ্ছে না | সেশন ম্যানেজার বন্ধ | সেশন ম্যানেজার সার্ভিস চালু আছে কিনা দেখুন |
| সেশন বারবার বিচ্ছিন্ন হচ্ছে | ফোনের ইন্টারনেট | ফোনের ইন্টারনেট সংযোগ পরীক্ষা করুন |
| "সেশন পাওয়া যায়নি" | সেশন মুছে ফেলা হয়েছে | নতুন সেশন তৈরি করুন |
