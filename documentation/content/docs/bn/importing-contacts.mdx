## CSV ফরম্যাট

নিচের কলাম সহ একটি CSV ফাইল আপলোড করুন:

| কলাম | আবশ্যক | বিবরণ |
|------|--------|--------|
| `name` | হ্যাঁ | কন্টাক্টের নাম |
| `phone` | হ্যাঁ | ফোন নম্বর (যেকোনো ফরম্যাট) |
| `email` | না | ইমেইল ঠিকানা |
| `*` | না | অতিরিক্ত কলাম `customFields` হিসেবে যুক্ত হবে |

### উদাহরণ CSV

```csv
name,phone,email,city,business
ফারহান আহমেদ,01712345001,farhan@test.com,ঢাকা,খুচরা বিক্রেতা
সুরাইয়া বেগম,+8801812345002,,নারায়ণগঞ্জ,পাইকার
```

## ফোন নর্মালাইজেশন

সব ফোন নম্বর স্বয়ংক্রিয়ভাবে E.164 ফরম্যাটে রূপান্তরিত হয়:

| ইনপুট | আউটপুট | নোট |
|-------|---------|------|
| `01712345001` | `+8801712345001` | বাংলাদেশ ডিফল্ট |
| `+8801812345002` | `+8801812345002` | ইতিমধ্যে E.164 |
| `017-1234-5001` | `+8801712345001` | ফরম্যাটিং মুছে দেয় |

অবৈধ নম্বর স্কিপ করা হয় এবং ইম্পোর্ট সামারিতে রিপোর্ট করা হয়।

## ডিডুপ্লিকেশন

কন্টাক্ট `(userId, phone)` দ্বারা ডিডুপ্লিকেট হয়। একই ফোনে আগে থেকে কন্টাক্ট থাকলে আপডেট হবে (upsert), ডুপ্লিকেট হবে না।

## API এন্ডপয়েন্ট

```
POST /api/v1/contacts/import/csv
Content-Type: multipart/form-data
Field: file (CSV ফাইল)
```

## অপ্ট-আউট

কন্টাক্ট অপ্ট-আউট করতে:

```
PATCH /api/v1/contacts/:id/opt-out
```

অপ্ট-আউট কন্টাক্ট **ভবিষ্যতের সব ক্যাম্পেইন থেকে বাদ** যাবে।
